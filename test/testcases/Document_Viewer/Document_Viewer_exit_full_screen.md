# 文档查看器：退出全屏方式

**前置条件**

1. 打开pdf格式文档
2. 打开djvu格式文档

**用例步骤**


| 编号 | 步骤                                             | 预期     |
| ---- | ------------------------------------------------ | -------- |
| 1    | 进入全屏，点击工具栏右侧退出全屏菜单             | 退出全屏 |
| 2    | 进入全屏，点击键盘esc按钮                        | 退出全屏 |
| 3    | 进入全屏，按快捷键F1 1                           | 退出全屏 |
| 4    | 进入全屏，右击文档弹出右键菜单，点击“退出全屏” | 退出全屏 |
