# 文档查看器：选中文本后右击



| 编号 | 步骤                                            | 预期                             |
| ---- | ----------------------------------------------- | -------------------------------- |
| 1    | 打开PDF文档，选中一段文本，鼠标在选中区域右击   | 弹出高亮相关的右键菜单           |
| 2    | 打开PDF文档，选中一段文本，鼠标在非选中区域右击 | 弹出进行上下页跳转相关的右键菜单 |
