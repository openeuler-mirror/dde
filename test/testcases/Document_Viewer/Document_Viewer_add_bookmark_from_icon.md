# 文档查看器：点击书签图标添加书签

前置条件

1.打开pdf格式文档
2.打开djvu格式文档

用例步骤


| 编号 | 步骤                                                   | 预期                                                                                 |
| ---- | ------------------------------------------------------ | ------------------------------------------------------------------------------------ |
| 1    | 进入缩略图\_书签页面，查看未添加任何书签时检查界面显示 | 界面显示正确                                                                         |
| 2    | 点击书签图标上方的添加书签按钮                         | 当前浏览的页面可以加入书签                                                           |
| 3    | 查看加入书签后的页面显示                               | 左边栏列出加入书签的页面：以页面缩略图和页面数（如页面20）形式显示；按照页面顺序展示 |
