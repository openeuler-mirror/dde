#  终端使用上下键查看输入历史命令

| 编号 | 步骤                                          | 预期                 |
| ---- | --------------------------------------------- | ------------------- |
| 1    | 打开终端，执行命令：#ls | 执行成功    |
| 2 |打开终端，执行命令：#lsblk  | 执行成功 |
| 3 |打开终端，执行命令：#free  | 执行成功 |
| 4 |通过键盘上下键查看历史命令 | 可以正常查看到以上历史命令 |
